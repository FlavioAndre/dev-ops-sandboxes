##### Installation
- Install NodeJS
- Install Grunt runnig:
```
npm install -g grunt-cli
npm install grunt --save-dev
```
- Include Grunt on your package.json file, as following:
```json
{
  "name": "my-project-name",
  "version": "0.1.0",
  "devDependencies": {
    "grunt": "~0.4.5"
  }
}
```
